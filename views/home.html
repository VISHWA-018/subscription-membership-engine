⚙️ How the Tier Engine Works

Every protected request flows through a 3-layer middleware chain

Layer 1: authMiddleware → Verify JWT token, extract user identity
Layer 2: expiryMiddleware → Check subscription validity, auto-expire if overdue
Layer 3: tierMiddleware(n) → Compare access_level ≥ required level & check download limits
✅ All checks pass → Controller serves content
